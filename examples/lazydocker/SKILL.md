# lazydocker

A simple terminal UI for both docker and docker-compose, written in Go with the [gocui](https://github.com/jroimartin/gocui 'gocui') library.

## When to Use

- Run `lazydocker` commands
- Make HTTP requests
- Style terminal output

## When NOT to Use

- GUI or web-based workflows where CLI is not available
- Projects using Python or JavaScript (different ecosystem)

## Quick Start

### Install

```bash
go install github.com/jesseduffield/lazydocker@latest
```

