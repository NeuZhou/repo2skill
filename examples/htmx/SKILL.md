*high power tools for HTML*

## When NOT to Use

## Quick Start

### Install

```bash
npm install htmx
```

### Basic Usage

```html
  <script src="https://cdn.jsdelivr.net/npm/htmx.org@2.0.8/dist/htmx.min.js"></script>
  <!-- have a button POST a click via AJAX -->
  <button hx-post="/clicked" hx-swap="outerHTML">
    Click Me
  </button>
```
