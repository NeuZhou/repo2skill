# trpc

tRPC allows you to easily build & consume fully typesafe APIs without schemas or code generation.

## When to Use

- Make HTTP requests
- Interact with REST APIs
- Run or write tests
- Build command-line interfaces

## When NOT to Use

- Projects using Python or Rust (different ecosystem)
