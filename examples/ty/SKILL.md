# ty

An extremely fast Python type checker and language server, written in Rust.

## When to Use

- Build web servers or APIs with ty

## When NOT to Use

- Projects using TypeScript or Rust (different ecosystem)

## Quick Start

### Install

To install ty, see the [installation](https://docs.astral.sh/ty/installation/) documentation.

To add the ty language server to your editor, see the [editor integration](https://docs.astral.sh/ty/editors/) guide.

### Basic Usage

```shell
uvx ty check
```

## Key Features

- 10x - 100x faster than mypy and Pyright
- Comprehensive diagnostics with rich contextual information
- Configurable rule levels, per-file overrides, suppression comments, and first-class project support
- Designed for adoption, with support for redeclarations and partially typed code
- Language server with code navigation, completions, code actions, auto-import, inlay hints, on-hover help, etc.

- `annotations`
