# Changelog

All notable changes to repo2skill will be documented in this file.

## [0.1.0] - 2026-03-16

### Added
- Initial release
- Convert GitHub repos to OpenClaw SKILL.md
- 20+ language support (TypeScript, Python, Rust, Go, Java, C#, and more)
- Framework detection (MCP servers, AI agents, web frameworks, CLI tools)
- Quality scoring system
- Monorepo detection (npm/yarn/pnpm workspaces, Lerna, Nx, Turborepo)
- CLI with `scan`, `validate`, `lint`, `compare`, `template` commands
- README parser for structured extraction
- JSON/YAML output formats
- 100+ tests
