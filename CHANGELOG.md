# Changelog

All notable changes to repo2skill will be documented in this file.

## [3.5.0] - 2026-03-16

### Added
- 20+ language support (Haskell, Lua, Zig, C/C++, R, and more)
- README parser for structured extraction
- Enhanced monorepo detection
- Full README rewrite
- 35 new tests

## [3.4.0] - 2026-03-15

### Added
- Linter diagnostics with AgentSkills spec validation
- Template scaffolding (`--type cli/mcp-server/web-api/library`)
- Enhanced interactive mode
- Changelog generator from git history

## [3.3.0] - 2026-03-14

### Added
- Skill dependency graph visualization (interactive HTML)
- Diff viewer for SKILL.md comparison
- Quality report generator
- Plugin system
- Enhanced update checker

## [3.2.0] - 2026-03-13

### Added
- Monorepo support (npm/yarn/pnpm workspaces, Lerna, Nx, Turborepo)
- Skill merging
- Multi-format output (JSON, YAML)
- Health check command
- Version history tracking

## [3.1.0] - 2026-03-12

### Added
- AI enhancement with LLM-powered descriptions
- ClawHub marketplace publishing
- Framework detection (MCP, AI agents, web frameworks)
- Skill testing and validation
- Stats dashboard

## [3.0.0] - 2026-03-11

### Added
- Major rewrite with performance optimizations
- GitHub API caching
- New documentation site
